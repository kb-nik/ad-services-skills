# Русская копия skill

Здесь лежит русская версия материалов из `../SKILL.md` и `../references/`.

Что важно:

- Основной рабочий skill для агента остаётся английским.
- Эта папка нужна как человекочитаемая копия для команды.
- Структура сохранена почти один в один, чтобы было легко сопоставлять файлы.

Состав:

- `SKILL.ru.md` — перевод основного skill
- `references/categories.ru.md` — категории сервисов
- `references/workflow.ru.md` — общий пайплайн копания
- `references/questions.ru.md` — универсальные вопросы к незнакомому сервису
- `references/web.ru.md` — подход к web/API сервисам
- `references/protocols-and-storage.ru.md` — брокеры, хранилища, SSH, воркеры
- `references/reverse.ru.md` — бинарные и реверсные сервисы
- `references/operations.ru.md` — live-round, preflight, incident mode, farm-ready validation
- `references/defense.ru.md` — минимальные фиксы и SLA
- `references/case-studies.ru.md` — паттерны из реальных кейсов

Если нужно, можно читать русский набор как основную документацию, а английский держать как боевую версию для самой модели.
